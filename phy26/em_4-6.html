麦克斯韦方程组（积分形式）

麦克斯韦将电磁学的基本规律总结为四个方程，构成了经典电磁理论的完整体系：

方程 I
高斯定理（电场）
$$\oiint_S \vec{D}\cdot d\vec{S} = \iiint_V \rho\,dV$$变化的电场有源，自由电荷是电场的源。
方程 II
高斯定理（磁场）
$$\oiint_S \vec{B}\cdot d\vec{S} = 0$$磁场无源，不存在磁单极子。
方程 III
法拉第感应定律
$$\oint_L \vec{E}\cdot d\vec{l} = -\iint_S \frac{\partial \vec{B}}{\partial t}\cdot d\vec{S}$$变化的磁场激发涡旋电场。
方程 IV
安培-麦克斯韦定律
$$\oint_L \vec{H}\cdot d\vec{l} = \iint_S \!\left(\vec{j}_c + \frac{\partial \vec{D}}{\partial t}\right)\!\cdot d\vec{S}$$传导电流和位移电流共同激发磁场。
麦克斯韦方程组（微分形式）

利用高斯散度定理和斯托克斯定理，可将积分形式转化为微分形式：

$$\nabla\cdot\vec{D} = \rho, \quad \nabla\cdot\vec{B} = 0$$ $$\nabla\times\vec{E} = -\frac{\partial\vec{B}}{\partial t}, \quad \nabla\times\vec{H} = \vec{j}_c + \frac{\partial\vec{D}}{\partial t}$$
辅助关系 在线性各向同性介质中：$\vec{D} = \varepsilon\vec{E} = \varepsilon_0\varepsilon_r\vec{E}$，$\vec{B} = \mu\vec{H} = \mu_0\mu_r\vec{H}$，$\vec{j}_c = \gamma\vec{E}$（欧姆定律微分形式）。
方程组的对称性与深刻含义
  • ①
    电磁对称性：方程 III 和 IV 揭示了电场与磁场的相互激发：$\partial\vec{B}/\partial t$ 激发 $\vec{E}$，$\partial\vec{D}/\partial t$ 激发 $\vec{H}$
  • ②
    预言电磁波：在无源区域（$\rho=0$，$\vec{j}_c=0$），方程组预言了以速度 $c = 1/\sqrt{\varepsilon_0\mu_0}$ 传播的电磁波
  • ③
    统一光与电磁：$c = 3\times10^8\,\text{m/s}$ 与光速完全一致，证明光是一种电磁波
  • ④
    相对论基础：麦克斯韦方程组在洛伦兹变换下保持不变，是狭义相对论的出发点之一
电磁波的产生与传播

在无源区域，麦克斯韦方程组给出波动方程：

$$\nabla^2\vec{E} = \varepsilon\mu\frac{\partial^2\vec{E}}{\partial t^2}, \quad \nabla^2\vec{H} = \varepsilon\mu\frac{\partial^2\vec{H}}{\partial t^2}$$

波速：$u = \dfrac{1}{\sqrt{\varepsilon\mu}} = \dfrac{c}{\sqrt{\varepsilon_r\mu_r}}$，真空中 $c = \dfrac{1}{\sqrt{\varepsilon_0\mu_0}} \approx 3\times10^8\,\text{m/s}$。

平面电磁波的解 沿 $x$ 方向传播的平面波：$E_y = E_0\cos(\omega t - kx)$，$H_z = H_0\cos(\omega t - kx)$，其中 $k = \omega/u = 2\pi/\lambda$。
电磁波的基本性质
  • 横
    横波：$\vec{E}$、$\vec{H}$ 均垂直于传播方向，且 $\vec{E}\perp\vec{H}$，三者构成右手系：$\vec{E}\times\vec{H}$ 指向传播方向
  • 同
    同相位：$\vec{E}$ 与 $\vec{H}$ 振动相位相同，同时达到最大值，同时为零
  • 比
    振幅关系：$\sqrt{\varepsilon}E_0 = \sqrt{\mu}H_0$，即 $E_0/H_0 = \sqrt{\mu/\varepsilon}$（波阻抗）；真空中 $E_0 = cB_0$
  • 能
    能量均分：电场能量密度 $\omega_e = \frac{1}{2}\varepsilon E^2$ 等于磁场能量密度 $\omega_m = \frac{1}{2}\mu H^2$，总能量密度 $\omega = \varepsilon E^2 = \mu H^2$
坡印廷矢量与辐射强度

电磁波携带能量，单位时间通过单位面积的能量（辐射强度）由坡印廷矢量描述：

$$\vec{S} = \vec{E}\times\vec{H}$$

$|\vec{S}|$ 的单位为 W/m²，方向为能量传播方向（即电磁波传播方向）。

平均辐射强度（光强）：$\bar{S} = \dfrac{1}{2}E_0 H_0 = \dfrac{E_0^2}{2\mu u} = \dfrac{1}{2}\varepsilon u E_0^2$

光压 电磁波照射到物体上会产生辐射压力（光压）。完全吸收时：$p = I/c$；完全反射时：$p = 2I/c$，其中 $I$ 为辐射强度（W/m²）。

例题 1：电磁波的能量与光压

太阳光在地球表面的辐射强度（太阳常数）约为 $I = 1.4\,\text{kW/m}^2$。求：(1) 地球表面电场振幅 $E_0$；(2) 太阳光对完全吸收表面的光压 $p$。

平均辐射强度：$I = \bar{S} = \dfrac{1}{2}\varepsilon_0 c E_0^2$，故：$$E_0 = \sqrt{\frac{2I}{\varepsilon_0 c}} = \sqrt{\frac{2\times1400}{8.85\times10^{-12}\times3\times10^8}} \approx 1027\,\text{V/m}$$
磁场振幅：$B_0 = E_0/c \approx 3.42\,\mu\text{T}$，$H_0 = B_0/\mu_0 \approx 2.72\,\text{A/m}$
光压（完全吸收）：$$p = \frac{I}{c} = \frac{1400}{3\times10^8} \approx 4.67\,\mu\text{Pa}$$
结果：$E_0 \approx 1027\,\text{V/m}$，$B_0 \approx 3.42\,\mu\text{T}$，光压 $p \approx 4.67\,\mu\text{Pa}$（极小，但对微粒和航天器有实际影响）。
例题 2：介质中的电磁波速度

玻璃的相对介电常数 $\varepsilon_r = 2.25$，相对磁导率 $\mu_r \approx 1$。求：(1) 光在玻璃中的传播速度；(2) 折射率 $n$；(3) 玻璃中电场与磁场的振幅关系。

传播速度：$$u = \frac{c}{\sqrt{\varepsilon_r\mu_r}} = \frac{3\times10^8}{\sqrt{2.25}} = 2\times10^8\,\text{m/s}$$
折射率：$n = c/u = \sqrt{\varepsilon_r\mu_r} = 1.5$（与实验值吻合）
振幅关系：$E_0/H_0 = \sqrt{\mu/\varepsilon} = \sqrt{\mu_0/(\varepsilon_0\times2.25)} \approx 251\,\Omega$
结果：$u = 2\times10^8\,\text{m/s}$，$n = 1.5$，波阻抗 $\approx 251\,\Omega$（真空波阻抗 $\approx 377\,\Omega$）。
