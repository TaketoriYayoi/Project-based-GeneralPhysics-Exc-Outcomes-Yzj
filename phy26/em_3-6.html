安培力公式

载流导线电流元 $Id\vec{l}$ 在磁场中所受的力：

$$d\vec{F} = Id\vec{l}\times\vec{B}$$

最简单情形（均匀磁场，直导线）：$F = BIl\sin\theta$，$\theta = \angle(\vec{l},\vec{B})$

安培力 vs 洛伦兹力 安培力本质是导体内所有自由电子受洛伦兹力的宏观合力。电流元 $Id\vec{l} = nqS\,dl\cdot\vec{v}$，代入洛伦兹力即可推出安培力公式。
交互仿真：安培力方向与大小（均匀磁场）

拖动滑块改变电流方向与磁场的夹角 $\theta$，直观看到安培力 $F\propto\sin\theta$。

θ = 90° F/F_max = 1.000
红：电流方向 I；蓝：磁场 B；绿：安培力 F = BIl sinθ
例2：斜置载流导线在无限长直导线旁的安培力

无限长直载流导线 $I_1$，同一平面内有导线（长 $L$，电流 $I_2$），导线左下角距 $I_1$ 为 $r$，与水平夹角 $\alpha$。求该导线受到的安培力。

磁场：$I_1$ 在距离 $x$ 处产生 $\vec{B} = \dfrac{\mu_0 I_1}{2\pi x}(-\hat{k})$（垂直纸面向内）
电流元：$I_2 d\vec{l} = I_2 dx\,\hat{i} + I_2\tan\alpha\, dx\,\hat{j}$
安培力元：$d\vec{F} = \dfrac{\mu_0 I_1 I_2}{2\pi x}dx(\hat{j} - \tan\alpha\,\hat{i})$
积分：从 $r$ 到 $r+L\cos\alpha$ 积分，得 $\vec{F} = \dfrac{\mu_0 I_1 I_2}{2\pi}(\hat{j}-\tan\alpha\,\hat{i})\ln\dfrac{r+L\cos\alpha}{r}$
结果：$\vec{F} = \dfrac{\mu_0 I_1 I_2}{2\pi}\ln\!\dfrac{r+L\cos\alpha}{r}\cdot(\hat{j}-\tan\alpha\,\hat{i})$
均匀磁场中弯曲载流导线的安培力

在均匀磁场 $\vec{B}$ 中，弯曲载流导线 $ab$ 所受的安培力等于等效直导线（连接 $a$、$b$ 两端的直线段 $\vec{L}$）所受的安培力：

$$\vec{F} = \int_L d\vec{F} = I\int_L d\vec{l}\times\vec{B} = I\vec{L}\times\vec{B}$$

其中 $\vec{L} = \overrightarrow{ab}$（从起点 $a$ 到终点 $b$ 的矢量），与导线的具体形状无关！

推论：闭合载流回路在均匀磁场中的合安培力为零 闭合回路 $\vec{L}_{ab} + \vec{L}_{ba} = \vec{0}$，故 $\vec{F}_{合} = I\vec{0}\times\vec{B} = \vec{0}$。但磁力矩一般不为零。
交互仿真：弯曲导线等效直导线受力

选择不同形状的弯曲导线（半圆弧、折线、正弦波形），观察安培力始终等于等效直导线受力。

黄色虚线：等效直导线 L；橙色箭头：安培力 F = ILBsinθ
载流线圈的磁矩与力矩

$N$ 匝面积为 $S$ 的载流线圈，磁矩：$\vec{P}_m = NIS\hat{n}$（$\hat{n}$ 由右手螺旋定则确定）

在匀强磁场 $\vec{B}$ 中受到磁力矩：

$$\vec{M} = \vec{P}_m\times\vec{B}$$

大小：$M = P_m B\sin\theta = NISB\sin\theta$，$\theta=\angle(\hat{n},\vec{B})$

  • ①
    $\theta=0$（$\vec{P}_m\parallel\vec{B}$）：$M=0$，稳定平衡（势能最低）
  • ②
    $\theta=\pi$（$\vec{P}_m$反平行 $\vec{B}$）：$M=0$，不稳定平衡（势能最高）
  • ③
    $\theta=\pi/2$：$M = M_{max} = NBIS$（力矩最大）
磁势能 $V_{mp} = -\vec{P}_m\cdot\vec{B} = -P_mB\cos\theta$；磁力矩趋向使 $\theta\to 0$，即线圈转到与 $\vec{B}$ 对齐的位置（势能最低）。
磁力的功

磁力对运动载流导线所做的功：$W = I\Delta\Phi_m$

即电流强度与磁通量增量之积，与路径无关（由于电流 $I$ 恒定）。

例题：圆形载流线圈在磁场中的力矩与功

半径 $R$ 的圆形线圈，电流 $I$，处于均匀磁场 $B$ 中，线圈平面与 $\vec{B}$ 平行（$\theta=\pi/2$）。（1）求以直径为轴的磁力矩；（2）线圈在力矩作用下转过 $90°$，求功。

磁矩：$\vec{P}_m = I\cdot\pi R^2\cdot(-\hat{k})$
力矩（1）：$M_{max} = P_m B = \pi R^2 IB$，方向由 $\vec{P}_m\times\vec{B}$ 确定
功（2）：$W = V_{初}-V_{末} = -P_mB\cos(\pi/2)-(-P_mB\cos 0) = \pi R^2 IB$
结果：（1）$M = \pi R^2 IB$；（2）$W = \pi R^2 IB$
运动电荷产生的磁场

从毕奥-萨伐尔定律推导单个运动电荷产生的磁场：

$$\vec{B} = \frac{\mu_0}{4\pi}\frac{q\vec{v}\times\hat{r}}{r^2}$$

推导：$I = nqSv$，元体积中粒子数 $dN = nS\,dl$，由 B-S 定律除以 $dN$ 即得单粒子磁场。

与电场类比 运动电荷的电场：$\vec{E} = \dfrac{q}{4\pi\varepsilon_0}\dfrac{\hat{r}}{r^2}$；磁场：$\vec{B} = \dfrac{\mu_0 q}{4\pi}\dfrac{\vec{v}\times\hat{r}}{r^2} = \dfrac{\vec{v}\times\vec{E}}{c^2}$（在低速极限下）。
这再次印证了磁场是电场的相对论效应。

磁场与电场的对称性
物理量 电场 磁场
源电荷 $q$运动电荷 $q\vec{v}$（电流）
场方程$\vec{E}=\frac{q\hat{r}}{4\pi\varepsilon_0 r^2}$$\vec{B}=\frac{\mu_0 q\vec{v}\times\hat{r}}{4\pi r^2}$
高斯定理$\nabla\cdot\vec{E}=\rho/\varepsilon_0\neq 0$$\nabla\cdot\vec{B}=0$
环路定理$\nabla\times\vec{E}=0$（静电场）$\nabla\times\vec{B}=\mu_0\vec{j}$
