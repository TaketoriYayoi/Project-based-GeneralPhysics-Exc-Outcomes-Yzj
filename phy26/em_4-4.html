自感线圈储存的磁场能量

RL 串联电路接通电源后，电流从 0 增大到稳态值 $I_0$。在此过程中，电源做的功一部分消耗在电阻上，另一部分以磁场能量的形式储存在自感线圈中。

对自感线圈两端电压 $U_L = L\,dI/dt$ 积分，得到储存的磁场能量：

$$W_m = \int_0^{I_0} U_L I\,dt = \int_0^{I_0} L I\,dI = \frac{1}{2}L I_0^2$$
类比电场能量 电容器储存电场能量 $W_e = \dfrac{1}{2}CV^2$，自感线圈储存磁场能量 $W_m = \dfrac{1}{2}LI^2$。两者形式完全对称：$C \leftrightarrow L$，$V \leftrightarrow I$。
磁场能量的本质

磁场能量储存在磁场本身中，而不是在导线中。以长直螺线管为例：

螺线管自感 $L = \mu_0 n^2 V$，通电流 $I$ 时，内部磁场 $B = \mu_0 n I$，磁场能量：

$$W_m = \frac{1}{2}LI^2 = \frac{1}{2}\mu_0 n^2 V \cdot I^2 = \frac{1}{2}\mu_0 n^2 I^2 \cdot V = \frac{B^2}{2\mu_0}\cdot V$$

这说明磁场能量均匀分布在螺线管内部体积 $V$ 中，单位体积的磁场能量（磁场能量密度）为：

$$\omega_m = \frac{W_m}{V} = \frac{B^2}{2\mu_0} = \frac{1}{2}\mu_0 H^2 = \frac{1}{2}BH$$
RL 电路的能量转化过程
  • ①
    充电阶段（接通电源）：电源做功 $W_{电源}$，一部分 $W_R = \int I^2 R\,dt$ 耗散在电阻，另一部分 $W_m = \frac{1}{2}LI_0^2$ 储存在磁场中
  • ②
    放电阶段（断开电源）：磁场能量 $W_m$ 全部转化为电阻热，$W_m = \int_0^\infty I^2 R\,dt$
  • ③
    时间常数：$\tau = L/R$，$\tau$ 越大，充放电越慢，磁场能量的建立与释放越缓慢
磁场能量密度公式

在磁导率为 $\mu$ 的介质中，磁场能量密度为：

$$\omega_m = \frac{1}{2}\mu H^2 = \frac{B^2}{2\mu} = \frac{1}{2}\vec{B}\cdot\vec{H}$$

对于非均匀磁场，总磁场能量为对全空间积分：

$$W_m = \iiint_V \omega_m\,dV = \iiint_V \frac{B^2}{2\mu}\,dV$$
与电场能量密度对比 电场能量密度：$\omega_e = \dfrac{1}{2}\varepsilon E^2 = \dfrac{D^2}{2\varepsilon} = \dfrac{1}{2}\vec{D}\cdot\vec{E}$
磁场能量密度：$\omega_m = \dfrac{1}{2}\mu H^2 = \dfrac{B^2}{2\mu} = \dfrac{1}{2}\vec{B}\cdot\vec{H}$
两者形式完全对称，体现了电磁场的对偶性。
磁场能量密度的分布特点
  • 均
    均匀磁场：$\omega_m$ 处处相同，$W_m = \omega_m \cdot V$。例如长直螺线管内部。
  • 非
    非均匀磁场：$\omega_m \propto B^2$，磁场越强的区域能量密度越大。例如同轴电缆中 $B \propto 1/r$，内侧能量密度更高。
  • 外
    外部磁场：即使在导体外部，只要有磁场存在，就有磁场能量。磁场能量弥散在整个空间。
计算磁场能量的两种方法 方法一：$W_m = \dfrac{1}{2}LI^2$（已知自感和电流时使用）
方法二：$W_m = \displaystyle\iiint \dfrac{B^2}{2\mu}\,dV$（已知磁场分布时使用，适合非均匀场）

例题 1：RL 电路的磁场能量

电源 $\varepsilon$、电阻 $R$、电感 $L$ 串联，$t=0$ 时接通电源（初始电流为零）。求：(1) 电流随时间的变化；(2) 稳态时磁场储存的能量；(3) 整个充电过程中电源做的功和电阻消耗的热量。

微分方程：由基尔霍夫电压定律，$\varepsilon = IR + L\dfrac{dI}{dt}$，初始条件 $I(0) = 0$。
求解电流：$$I(t) = \frac{\varepsilon}{R}\!\left(1 - e^{-t/\tau}\right), \quad \tau = \frac{L}{R}$$稳态电流 $I_0 = \varepsilon/R$。
磁场能量：稳态时 $W_m = \dfrac{1}{2}LI_0^2 = \dfrac{L\varepsilon^2}{2R^2}$
电源做功：$W_{电源} = \displaystyle\int_0^\infty \varepsilon I\,dt = \dfrac{L\varepsilon^2}{R^2}$（可验证：$W_{电源} = 2W_m$）
电阻热量：$Q = W_{电源} - W_m = \dfrac{L\varepsilon^2}{2R^2} = W_m$，即充电过程中电源做功的一半储存在磁场，一半耗散为热。
结论：充电过程中，电源做功 $W = \dfrac{L\varepsilon^2}{R^2}$，其中一半 $W_m = \dfrac{L\varepsilon^2}{2R^2}$ 储存在磁场，另一半转化为电阻热。
例题 2：螺线管的磁场能量密度

横截面积 $S$、半径 $R$ 的实心金属圆环，套着匝数 $N$、电流 $I$、磁导率 $\mu$ 的螺线管（螺线管半径远小于圆环半径 $R$）。求螺线管内的磁场能量密度 $\omega_m$ 和总磁场能量 $W_m$。

磁场强度：由安培环路定理，$\oint \vec{H}\cdot d\vec{l} = NI$，对半径为 $R$ 的环路：$H \cdot 2\pi R = NI$，故 $H = \dfrac{NI}{2\pi R}$
磁感应强度：$B = \mu H = \dfrac{\mu NI}{2\pi R}$
能量密度：$$\omega_m = \frac{1}{2}\mu H^2 = \frac{\mu N^2 I^2}{8\pi^2 R^2}$$
总能量：螺线管体积 $V = S \cdot 2\pi R$（截面积乘周长），故：$$W_m = \omega_m \cdot V = \frac{\mu N^2 I^2}{8\pi^2 R^2}\cdot 2\pi RS = \frac{\mu N^2 I^2 S}{4\pi R}$$
结果：$\omega_m = \dfrac{\mu N^2 I^2}{8\pi^2 R^2}$，$W_m = \dfrac{\mu N^2 I^2 S}{4\pi R}$。可验证 $W_m = \dfrac{1}{2}LI^2$，其中 $L = \dfrac{\mu N^2 S}{2\pi R}$。
