静电能的来源：组装电荷系需做功

将各点电荷从无穷远处逐个移到指定位置，外力克服（或顺应）库仑力所做的功，储存为系统的静电势能（电能）。由于库仑力是保守力，这个能量与移动路径无关，只取决于最终的电荷分布。

关键思想：第一个电荷从无穷远移来无需做功（无其他电荷产生场）；第二个电荷需克服第一个电荷的库仑力；第三个电荷需克服前两个……以此类推。

两个点电荷的互能

设 $q_1$ 已在原位，将 $q_2$ 从无穷远移至与 $q_1$ 相距 $r_{12}$ 处，外力做功（即系统获得的电能）为：

$$W = q_2 V_1(r_{12}) = q_2 \cdot \frac{q_1}{4\pi\varepsilon_0 r_{12}} = \frac{q_1 q_2}{4\pi\varepsilon_0 r_{12}}$$

由对称性，也可写成 $W = q_1 V_2(r_{12})$。取两者平均，得到与书写顺序无关的对称形式：

$$\boxed{W_{12} = \frac{1}{2}(q_1 V_{12} + q_2 V_{21}) = \frac{q_1 q_2}{4\pi\varepsilon_0 r_{12}}}$$
$n$ 个点电荷系的总电能

将 $n$ 个点电荷逐一从无穷远移入，总电能等于所有不重复对的互能之和：

$$W_e = \sum_{i等价地，对每个电荷 $q_i$，其所在处其他所有电荷产生的电势为 $V_i = \displaystyle\sum_{j\neq i}\frac{q_j}{4\pi\varepsilon_0 r_{ij}}$，则：

$$\boxed{W_e = \frac{1}{2}\sum_{i=1}^n q_i V_i}$$

系数 $\tfrac{1}{2}$ 是因为每对互能在双重求和中被计算了两次。

  • 注①
    $W_e$ 可正可负：同号电荷系（$W_e>0$）组装需外力做正功，电荷间相互排斥；异号电荷系（$W_e<0$）自发趋向聚合，需外力阻止。
  • 注②
    不含自能：点电荷本身的"自能"（将电荷从分散状态组装成点电荷所需的能量）发散，上式仅计算各点电荷之间的互能，与组装过程无关。
  • 注③
    连续分布推广：$W_e = \dfrac{1}{2}\displaystyle\int \rho V\,\mathrm{d}V$（体电荷）；面电荷类似，$W_e = \dfrac{1}{2}\displaystyle\int \sigma V\,\mathrm{d}S$。
三电荷系：$W_e = \frac{1}{2}(q_1V_1+q_2V_2+q_3V_3)$，共三对互能
电容器充电做功 = 储存的电能

对电容器充电时，将微小电荷元 $\mathrm{d}q'$ 从负极搬至正极，需克服两极板间电压 $v = q'/C$ 做功：

$$\mathrm{d}W = v\,\mathrm{d}q' = \frac{q'}{C}\mathrm{d}q'$$

充电从 $0$ 到 $Q$，总功（即储存的电能）：

$$W_e = \int_0^Q \frac{q'}{C}\mathrm{d}q' = \frac{Q^2}{2C}$$
$$\boxed{W_e = \frac{Q^2}{2C} = \frac{1}{2}QV = \frac{1}{2}CV^2}$$

三种形式等价（$Q=CV$），分别以电量、电压或两者表示。

三种表达形式的适用场合
  • $Q^2/2C$
    已知电量 $Q$ 和电容 $C$（如孤立导体或串联电路中电量守恒的情形），适合分析电容改变时能量变化。
  • $CV^2/2$
    已知电压 $V$ 和电容 $C$（如并联电路中电压守恒的情形），适合分析电容改变时能量变化。
  • $QV/2$
    已知 $Q$ 和 $V$，中间形式，常用于直接计算或验证。注意系数 $\tfrac{1}{2}$——它来自充电过程中电压从 $0$ 线性增大到 $V$，平均值为 $V/2$。
常见错误 充电完成后电荷量为 $Q$、电压为 $V$，但能量不是 $QV$，而是 $\frac{1}{2}QV$。多出的 $\frac{1}{2}$ 是因为充电初期电压低、末期电压高，做功是电压对电量的积分，不能直接相乘。
充电过程：$v$-$q$ 图线下面积 $= W_e = \frac{1}{2}QV$（三角形面积）
电场能量密度

以平行板电容器为例推导能量密度。设极板面积 $S$、间距 $d$、介质 $\varepsilon_r$，极板间匀强场 $E$，则：

$$C = \frac{\varepsilon_0\varepsilon_r S}{d}, \quad V = Ed, \quad W_e = \frac{1}{2}CV^2 = \frac{1}{2}\cdot\frac{\varepsilon_0\varepsilon_r S}{d}\cdot(Ed)^2 = \frac{1}{2}\varepsilon_0\varepsilon_r E^2 \cdot Sd$$

而极板间体积 $\mathcal{V} = Sd$，故单位体积能量（能量密度）：

$$\boxed{w_e = \frac{W_e}{\mathcal{V}} = \frac{1}{2}\varepsilon_0\varepsilon_r E^2 = \frac{1}{2}\varepsilon E^2 = \frac{1}{2}\vec{D}\cdot\vec{E}}$$

其中 $\varepsilon = \varepsilon_0\varepsilon_r$ 为介质的介电常数。最后一种写法 $\frac{1}{2}\vec{D}\cdot\vec{E}$ 对各向异性介质也成立（此时 $\vec{D}$ 与 $\vec{E}$ 方向一般不同）。

深刻含义：电能储存在电场中 能量密度 $w_e$ 的表达式表明，静电能储存在电场所占据的空间中，而非储存在电荷上。哪里有电场，哪里就有能量，密度为 $\frac{1}{2}\varepsilon E^2$。这是场论观点的核心体现，在电磁波（光）的传播中尤为重要。
总电场能量

对非均匀场，将全空间分成无穷多体积元，每个体积元 $\mathrm{d}\mathcal{V}$ 的能量为 $w_e\,\mathrm{d}\mathcal{V}$，积分得总电场能量：

$$\boxed{W_e = \int_{\text{全空间}} w_e\,\mathrm{d}\mathcal{V} = \int_{\text{全空间}} \frac{1}{2}\varepsilon E^2\,\mathrm{d}\mathcal{V}}$$
  • 意义
    $E$ 大的区域贡献更多能量（$w_e \propto E^2$）；场强集中的区域（如电容器内部、导体尖端附近）是能量密集区。
  • 等价
    对任意静电系统，$\displaystyle\int\frac{1}{2}\varepsilon E^2\,\mathrm{d}\mathcal{V} = \frac{1}{2}\sum_i q_i V_i$（两种计算总能量的方式完全等价，但积分形式更具场论意义）。
能量密度分布：$w_e = \frac{1}{2}\varepsilon E^2$，场强越大的区域颜色越亮
例  能量法推导球形电容器的电容

球形电容器：内球半径 $R_1$，外球壳内半径 $R_2$，两球间充满 $\varepsilon_r$ 的均匀电介质，内球带电量 $Q$。用能量法求电容 $C$。

球形电容器：场仅存在于 $R_1 < r < R_2$ 的球壳形区域，以此积分电场能量
① 求介质中的场强 $E(r)$：
由 $\vec{D}$ 的高斯定理，取半径 $r$（$R_1 < r < R_2$）球面： $$D\cdot 4\pi r^2 = Q \implies E = \frac{D}{\varepsilon_0\varepsilon_r} = \frac{Q}{4\pi\varepsilon_0\varepsilon_r r^2}$$ $r < R_1$ 及 $r > R_2$ 区域：$E = 0$（导体内部及外球壳屏蔽外部）。
② 计算总电场能量 $W_e$：
取厚度 $\mathrm{d}r$ 的球壳体积元 $\mathrm{d}\mathcal{V} = 4\pi r^2\,\mathrm{d}r$： $$W_e = \int_{R_1}^{R_2} \frac{1}{2}\varepsilon_0\varepsilon_r E^2 \cdot 4\pi r^2\,\mathrm{d}r = \int_{R_1}^{R_2}\frac{1}{2}\varepsilon_0\varepsilon_r \cdot\frac{Q^2}{(4\pi\varepsilon_0\varepsilon_r)^2 r^4}\cdot 4\pi r^2\,\mathrm{d}r$$ $$= \frac{Q^2}{8\pi\varepsilon_0\varepsilon_r}\int_{R_1}^{R_2}\frac{\mathrm{d}r}{r^2} = \frac{Q^2}{8\pi\varepsilon_0\varepsilon_r}\left(\frac{1}{R_1}-\frac{1}{R_2}\right) = \frac{Q^2(R_2-R_1)}{8\pi\varepsilon_0\varepsilon_r R_1 R_2}$$
③ 由 $W_e = Q^2/(2C)$ 反推电容： $$\frac{Q^2}{2C} = \frac{Q^2(R_2-R_1)}{8\pi\varepsilon_0\varepsilon_r R_1 R_2}$$ $$\frac{1}{2C} = \frac{R_2-R_1}{8\pi\varepsilon_0\varepsilon_r R_1 R_2} \implies C = \frac{4\pi\varepsilon_0\varepsilon_r R_1 R_2}{R_2-R_1}$$
答： $$C = \frac{4\pi\varepsilon_0\varepsilon_r R_1 R_2}{R_2-R_1}$$ 与 §2-3 用高斯定理直接积分电势差所得结果完全一致，验证了能量法的正确性。
能量法的优越性 能量法无需求电势差，直接对场能积分，再与 $W_e=Q^2/(2C)$ 对比即得 $C$。对几何复杂的电容器，能量法有时比积分电场更简便，是电磁学中"能量守恒"与"场的概念"相结合的典型应用。
