电容的定义

实验表明，孤立导体所带电量 $q$ 与其电势 $V$（以无穷远为零势点）的比值是一个与电量无关的常数，仅由导体的几何形状和尺寸决定。定义此比值为导体的电容（capacitance）：

$$C = \frac{q}{V}$$
  • 单位
    电容的 SI 单位为法拉（Farad，F）：$1\ \mathrm{F} = 1\ \mathrm{C/V}$。
    法拉是很大的单位，实用中常用 $\mu\mathrm{F}\ (10^{-6}\ \mathrm{F})$、$\mathrm{nF}\ (10^{-9}\ \mathrm{F})$、$\mathrm{pF}\ (10^{-12}\ \mathrm{F})$。
  • 本质
    $C$ 描述导体储存电荷的能力：$C$ 越大，在相同电势下能储存更多电荷；或在储存相同电荷时产生更低电势。
  • 关键
    $C$ 与是否带电无关——不论 $q$ 为零还是非零，$C$ 均由导体几何形状唯一决定。这是电容作为"固有属性"的核心。
孤立导体球的电容

设孤立导体球半径为 $R$，带电量为 $q$，以无穷远处为零势点。由高斯定理，球外场强与点电荷相同，球面（及内部）电势为：

$$V = \frac{1}{4\pi\varepsilon_0}\cdot\frac{q}{R}$$

由电容定义：

$$\boxed{C = \frac{q}{V} = 4\pi\varepsilon_0 R}$$
  • 结论
    $C = 4\pi\varepsilon_0 R$ 仅与半径 $R$ 有关，与电量 $q$ 无关。球越大，电容越大——相同电量下，大球面积更大，电荷分散，电势更低，故储电能力更强。
  • 数值
    地球半径 $R_\oplus \approx 6.4\times10^6\ \mathrm{m}$，其电容： $$C_\oplus = 4\pi\varepsilon_0 R_\oplus \approx 4\pi\times8.85\times10^{-12}\times6.4\times10^6 \approx 7.1\times10^{-4}\ \mathrm{F} \approx 710\ \mu\mathrm{F}$$ 仅约 $710\ \mu\mathrm{F}$，远小于工程电容器（可达 $\mathrm{F}$ 量级），说明孤立导体的电容很小。
孤立导体球：$C = 4\pi\varepsilon_0 R$，仅由半径决定，与带电量无关
为何 $C$ 与带电量无关？

从物理图像理解：当导体带电量从 $q$ 增大到 $2q$ 时，由线性叠加原理，电势也从 $V$ 增大到 $2V$，比值 $q/V$ 保持不变。这是静电场线性方程（泊松方程/拉普拉斯方程）的必然结果——场量与源量成正比。

与材料无关 对于孤立导体球，$C = 4\pi\varepsilon_0 R$ 中不含介电常数，因为孤立导体球周围是真空（或空气）。若导体球嵌入无限大均匀介质（介电常数 $\varepsilon_r$）中，则 $C = 4\pi\varepsilon_0\varepsilon_r R$，电容增大 $\varepsilon_r$ 倍。
推广：任意形状孤立导体 虽然 $C = 4\pi\varepsilon_0 R$ 仅适用于球形导体，但对任意形状孤立导体，电容仍定义为 $C = q/V$，且依然与带电量无关。形状决定电荷分布的形式，从而决定 $V/q$ 的比值，即电容。
电容器的结构与定义

电容器（capacitor）由两个靠得很近、中间由电介质隔开的金属导体（极板）组成。其核心特点是：将电场集中在两极板之间的有限空间内，极板外几乎无电场（理想情况）。

设两极板分别带电 $+q$ 和 $-q$（等量异号），两板间电势差为 $\Delta V = V_+ - V_-$，则电容定义为：

$$C = \frac{q}{\Delta V}$$
与孤立导体电容的区别 孤立导体：$C=q/V$（$V$ 相对无穷远）。电容器：$C=q/\Delta V$（$\Delta V$ 为两板间电势差）。电容器的优势在于两板相互感应，使每板能在更低电势差下储存更多电荷，$C$ 远大于孤立导体。
例 1  圆柱形（同轴）电容器

内筒半径 $R_A$，外筒半径 $R_B$，长度 $l$（$l\gg R_B$），两筒间充满相对介电常数 $\varepsilon_r$ 的电介质。内筒带电 $+q$，外筒带电 $-q$。求电容 $C$。

同轴圆柱：内筒 $+q$，外筒 $-q$，介质中电场沿径向，$R_A < r < R_B$
① 用 $\vec{D}$ 的高斯定理求 $D$：
取半径 $r$（$R_A < r < R_B$）、长度 $l$ 的同轴圆柱面为高斯面： $$D\cdot 2\pi r l = q \implies D = \frac{q}{2\pi r l}$$ 方向沿径向向外。
② 求场强 $E$： $$E = \frac{D}{\varepsilon_0\varepsilon_r} = \frac{q}{2\pi\varepsilon_0\varepsilon_r r l}$$
③ 求两板间电势差 $\Delta U$： $$\Delta U = V_A - V_B = \int_{R_A}^{R_B} E\,\mathrm{d}r = \frac{q}{2\pi\varepsilon_0\varepsilon_r l}\int_{R_A}^{R_B}\frac{\mathrm{d}r}{r} = \frac{q}{2\pi\varepsilon_0\varepsilon_r l}\ln\frac{R_B}{R_A}$$
④ 求电容： $$\boxed{C = \frac{q}{\Delta U} = \frac{2\pi\varepsilon_0\varepsilon_r l}{\ln(R_B/R_A)}}$$
答： $C = \dfrac{2\pi\varepsilon_0\varepsilon_r l}{\ln(R_B/R_A)}$。$l$ 越长、两筒半径比越接近1，电容越大。
例 2  平行板电容器（等量异号）

两块面积为 $S$、间距为 $d$ 的平行极板，中间填充相对介电常数 $\varepsilon_r$ 的均匀电介质，两板带 $\pm q$（等量异号）。求电容。

① 由高斯定理：两板间 $D = \sigma_0 = q/S$，$E = D/(\varepsilon_0\varepsilon_r) = q/(\varepsilon_0\varepsilon_r S)$。
② 电势差：$\Delta V = Ed = qd/(\varepsilon_0\varepsilon_r S)$。
③ 电容： $$\boxed{C = \frac{q}{\Delta V} = \frac{\varepsilon_0\varepsilon_r S}{d}}$$ 面积越大（极板重叠面积大）、间距越小、介电常数越大，电容越大。
答：$C = \varepsilon_0\varepsilon_r S/d$。
例 2 变体  平行板两极板电量不等（$+q$ 与 $-3q$）

若两平行板分别带电 $+q$ 和 $-3q$（不等量），此时不满足电容器的等量异号条件。设四个表面（从左到右）的面电荷密度对应电量分别为 $q_1,q_2,q_3,q_4$（左板左面、左板右面、右板左面、右板右面）。

四个表面电荷量 $q_1,q_2,q_3,q_4$；外侧 $q_1=q_4$，内侧 $q_2=-q_3$
四个方程的由来：
$q_1 + q_2 = q$（左板总电荷守恒）
$q_3 + q_4 = -3q$（右板总电荷守恒）
$q_2 = -q_3$（两板内侧等量异号——由内侧区域高斯定理，平行板间电场由内侧两面决定，静电平衡要求内侧必须等量异号）
$q_1 = q_4$（两板外侧面电荷相等）——推导如下：

每个无限大带电面在两侧产生场强大小 $|\sigma|/(2\varepsilon_0)$，取向右为正。
在左板内部（$\sigma_1$ 右侧，$\sigma_2,\sigma_3,\sigma_4$ 左侧），总场为零： $$\frac{\sigma_1 - \sigma_2 - \sigma_3 - \sigma_4}{2\varepsilon_0} = 0 \implies \sigma_1 = \sigma_2 + \sigma_3 + \sigma_4 \quad\cdots(1)$$ 在右板内部（$\sigma_1,\sigma_2,\sigma_3$ 右侧，$\sigma_4$ 左侧），总场为零： $$\frac{\sigma_1 + \sigma_2 + \sigma_3 - \sigma_4}{2\varepsilon_0} = 0 \implies \sigma_4 = \sigma_1 + \sigma_2 + \sigma_3 \quad\cdots(2)$$ $(1)+(2)$：$\sigma_1 + \sigma_4 = \sigma_1 + \sigma_4$（恒等），$(1)-(2)$：$\sigma_1 - \sigma_4 = -(\sigma_1 - \sigma_4)$，故 $\sigma_1 = \sigma_4$，即 $q_1 = q_4$。
这是静电平衡条件（导体内部场为零）对两板外侧的必然约束，反映了无限大平行板系统的对称性。
联立求解：
由 $q_2=-q_3$ 及两板守恒方程相加：$(q_1+q_2)+(q_3+q_4)=q+(-3q)=-2q$
$\Rightarrow q_1+q_4=-2q$
由 $q_1=q_4$：$2q_1=-2q \Rightarrow q_1=q_4=-q$
$q_2=q-q_1=q-(-q)=2q$
$q_3=-q_2=-2q$
结论：内侧 $q_2=2q$，$q_3=-2q$（等量异号，形成电容器效果）；
外侧 $q_1=q_4=-q\neq 0$，极板外侧存在电场，对应面电荷密度 $\sigma_{\text{外}}=-q/S$，场强 $E_{\text{外}}=q/(\varepsilon_0 S)$。
由于外侧有电场，此系统不满足电容器的定义（电场未被集中在两板之间），不能视为电容器。
答： $q_1=q_4=-q$，$q_2=2q$，$q_3=-2q$。外侧有电场 $E_{\text{外}}=q/(\varepsilon_0 S)$，系统不满足电容器条件。
一般结论 对两平行板（带电量 $Q_1$，$Q_2$）：外侧面电荷量均为 $(Q_1+Q_2)/2$，内侧面电荷量均为 $(Q_1-Q_2)/2$（等量异号）。仅当 $Q_1=-Q_2$ 时外侧才无电场，成为真正的电容器。
例 3  球形电容器

内球半径 $R_A$，外球壳内半径 $R_B$，两球间充满 $\varepsilon_r$ 的电介质，内球带 $+q$，外球壳带 $-q$。求电容。

球形电容器：内球 $+q$，外壳 $-q$，径向电场集中在 $R_A < r < R_B$
① 求 $D$、$E$（$R_A < r < R_B$）： 取半径 $r$ 球面为高斯面：$D\cdot 4\pi r^2 = q$，故 $$E = \frac{D}{\varepsilon_0\varepsilon_r} = \frac{q}{4\pi\varepsilon_0\varepsilon_r r^2}$$
② 求电势差： $$\Delta V = V_A - V_B = \int_{R_A}^{R_B}E\,\mathrm{d}r = \frac{q}{4\pi\varepsilon_0\varepsilon_r}\int_{R_A}^{R_B}\frac{\mathrm{d}r}{r^2} = \frac{q}{4\pi\varepsilon_0\varepsilon_r}\left(\frac{1}{R_A}-\frac{1}{R_B}\right) = \frac{q(R_B-R_A)}{4\pi\varepsilon_0\varepsilon_r R_A R_B}$$
③ 求电容： $$\boxed{C = \frac{q}{\Delta V} = \frac{4\pi\varepsilon_0\varepsilon_r R_A R_B}{R_B - R_A}}$$
答：$C = \dfrac{4\pi\varepsilon_0\varepsilon_r R_A R_B}{R_B-R_A}$。当 $R_B\to\infty$ 时，$C\to 4\pi\varepsilon_0\varepsilon_r R_A$，退化为孤立导体球（外球壳在无穷远）。
例 4  两平行长直导线的单位长度电容

两根半径为 $a$ 的无限长平行直导线，中心间距为 $d$（$d\gg a$），分别带线电荷密度 $+\lambda$ 和 $-\lambda$（因 $d\gg a$ 可视为均匀分布）。求单位长度电容 $C/l$。

两平行导线：$+\lambda$ 和 $-\lambda$，$d\gg a$；两线间电场叠加
① 各导线产生的场强：
距 $+\lambda$ 导线轴线距离 $r$ 处，由高斯定理：$E_+ = \lambda/(2\pi\varepsilon_0 r)$（径向向外）。
距 $-\lambda$ 导线轴线距离 $r'$ 处：$E_- = \lambda/(2\pi\varepsilon_0 r')$（径向向内，即指向 $-\lambda$）。
两线连线上（$a \leq x \leq d-a$，$x$ 为离 $+\lambda$ 轴的距离），两场同向（均从 $+\lambda$ 指向 $-\lambda$）： $$E = E_+ + E_- = \frac{\lambda}{2\pi\varepsilon_0 x} + \frac{\lambda}{2\pi\varepsilon_0(d-x)}$$
② 求两导线间电势差 $\Delta V$： $$\Delta V = \int_a^{d-a} E\,\mathrm{d}x = \frac{\lambda}{2\pi\varepsilon_0}\int_a^{d-a}\left(\frac{1}{x}+\frac{1}{d-x}\right)\mathrm{d}x$$ $$= \frac{\lambda}{2\pi\varepsilon_0}\left[\ln x - \ln(d-x)\right]_a^{d-a} = \frac{\lambda}{2\pi\varepsilon_0}\left[\ln\frac{d-a}{a} - \ln\frac{a}{d-a}\right]$$ $$= \frac{\lambda}{2\pi\varepsilon_0}\cdot 2\ln\frac{d-a}{a} = \frac{\lambda}{\pi\varepsilon_0}\ln\frac{d-a}{a}$$
③ 求单位长度电容： 单位长度的电量为 $\lambda$，故： $$\boxed{\frac{C}{l} = \frac{\lambda/l}{\Delta V/l}\cdot l = \frac{\lambda}{\Delta V} = \frac{\pi\varepsilon_0}{\ln\!\dfrac{d-a}{a}}}$$
答： 单位长度电容 $C/l = \dfrac{\pi\varepsilon_0}{\ln\!\dfrac{d-a}{a}}$。$d/a$ 越大（两线越远），电容越小；$d/a\to 1+$（两线紧贴）时，电容趋于无穷大。
串联（Series）

多个电容器首尾相接，每个电容器上的电量相同（中间节点电荷守恒），电压相加：

$$\frac{1}{C} = \frac{1}{C_1} + \frac{1}{C_2} + \cdots + \frac{1}{C_n}$$
  • 由来
    串联时 $q_1=q_2=\cdots=q$（中间节点孤立，正负感应电荷相消），总电压 $V=V_1+V_2+\cdots$，故 $V/q = \sum V_i/q = \sum 1/C_i$，即 $1/C = \sum 1/C_i$。
  • 特点
    串联等效电容小于最小单个电容；耐压能力提升；各电容分压 $V_i = q/C_i$，$C$ 小的分压多。
并联（Parallel）

多个电容器两端分别连在一起，每个电容器上的电压相同，电量相加：

$$C = C_1 + C_2 + \cdots + C_n$$
  • 由来
    并联时各电容两端电压相同 $V_1=V_2=\cdots=V$，总电量 $q=q_1+q_2+\cdots=C_1 V+C_2 V+\cdots=(C_1+C_2+\cdots)V$，故 $C=q/V=\sum C_i$。
  • 特点
    并联等效电容大于最大单个电容；储电量增大；各电容分配电量 $q_i=C_i V$，$C$ 大的存电多。
例  混联电路 $C_1\parallel C_2$ 后串联 $C_3$

$C_1=10\ \mu\mathrm{F}$，$C_2=5\ \mu\mathrm{F}$，$C_3=5\ \mu\mathrm{F}$，$V_{AB}=100\ \mathrm{V}$。

电路结构：从 $A$ 出发线路分叉，$C_1$、$C_2$ 并联后汇合，再串联 $C_3$ 到 $B$。

$C_1\parallel C_2$ 串联 $C_3$；$V_{AB}=100\ \mathrm{V}$
（1）AB 两端等效电容：
$C_1$ 与 $C_2$ 并联： $$C_{12} = C_1 + C_2 = 10 + 5 = 15\ \mu\mathrm{F}$$ $C_{12}$ 再与 $C_3$ 串联： $$\frac{1}{C} = \frac{1}{C_{12}} + \frac{1}{C_3} = \frac{1}{15} + \frac{1}{5} = \frac{1}{15} + \frac{3}{15} = \frac{4}{15}$$ $$\boxed{C = \frac{15}{4} = 3.75\ \mu\mathrm{F}}$$
（2）$C_2$ 上的电量与电压：
总电量（串联部分电量相同）： $$Q = C \cdot V_{AB} = 3.75\times10^{-6}\times100 = 3.75\times10^{-4}\ \mathrm{C} = 375\ \mu\mathrm{C}$$ $C_{12}$（即并联组）上的电量也为 $Q=375\ \mu\mathrm{C}$，两端电压： $$V_{12} = \frac{Q}{C_{12}} = \frac{375}{15} = 25\ \mathrm{V}$$ $C_2$ 与 $C_{12}$ 并联，两端电压相同： $$\boxed{V_2 = 25\ \mathrm{V}}, \quad Q_2 = C_2 V_2 = 5\times25 = 125\ \mu\mathrm{C}$$ 验证：$Q_1=C_1 V_1=10\times25=250\ \mu\mathrm{C}$，$Q_1+Q_2=375\ \mu\mathrm{C}\ \checkmark$
（3）$C_1$ 被击穿后，$C_3$ 上的电量与电压：
$C_1$ 击穿后相当于短路（击穿 = 导体通路），$C_1$ 所在支路变为导线，并联组 $C_1\parallel C_2$ 中 $C_1$ 短路，则$C_2$ 两端也被短路（电压被强制为零）——电荷全部释放，$C_2$ 退出电路。
等效电路变为：仅 $C_3$ 单独接在 $A$、$B$ 之间： $$Q_3 = C_3\cdot V_{AB} = 5\times100 = 500\ \mu\mathrm{C}$$ $$\boxed{Q_3 = 500\ \mu\mathrm{C},\quad V_3 = V_{AB} = 100\ \mathrm{V}}$$
答： （1）$C_{eq}=3.75\ \mu\mathrm{F}$；
（2）$Q_2=125\ \mu\mathrm{C}$，$V_2=25\ \mathrm{V}$；
（3）$C_1$ 击穿后 $C_2$ 被短路，$Q_3=500\ \mu\mathrm{C}$，$V_3=100\ \mathrm{V}$。
击穿的效果 电容器击穿后成为短路（两极板间导通，相当于一段导线）。与之并联的其他电容器也随之被短路（两端接在同一导线上，电压为零，电荷全部流走）。这与熔断器（开路）恰好相反。
