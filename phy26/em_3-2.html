毕奥-萨伐尔定律

电流元 $Id\vec{l}$ 在空间某点 P 处产生的磁感应强度为：

$$d\vec{B} = \frac{\mu_0}{4\pi} \frac{Id\vec{l} \times \hat{e}_r}{r^2}$$

大小：$dB = \dfrac{\mu_0}{4\pi}\dfrac{I\,dl\sin\theta}{r^2}$，其中 $\theta$ 为 $Id\vec{l}$ 与位矢 $\vec{r}$ 的夹角。

方向：$Id\vec{l}$ 转向 $\vec{r}$ 的右手螺旋前进方向（即 $Id\vec{l}\times\vec{r}$ 方向）。

真空磁导率：$\mu_0 = 4\pi\times10^{-7}\ \text{N·A}^{-2}$

一般求解步骤
  • 1
    取积分元：在导线上任取电流元 $Id\vec{l}$
  • 2
    写 $d\vec{B}$：由毕奥-萨伐尔定律写出微元磁场
  • 3
    分解分量：利用对称性分析各分量，垂直分量往往相消
  • 4
    积分：对有贡献的分量沿整个导线积分，合并得结果

例1：载流圆线圈轴线上的磁场

已知圆形线圈半径 $R$，通电流 $I$，求轴线上距圆心 $O$ 为 $x$ 处 P 点的磁感应强度。

取元：取圆上电流元 $Id\vec{l} = Idl\,\hat{k}$，P 点位矢 $\vec{r} = x\hat{i} - R\hat{j}$，$r = \sqrt{x^2+R^2}$
叉积：$d\vec{B} = \dfrac{\mu_0}{4\pi}\dfrac{Id\vec{l}\times\vec{r}}{r^3} = \dfrac{\mu_0 I\,dl}{4\pi r^3}(x\hat{j}+R\hat{i})$
对称性：$y$（横向）分量 $dB_y$ 绕轴积分相消，只剩 $x$ 方向
积分：$B = \displaystyle\int dB_x = \frac{\mu_0 I R^2}{2(x^2+R^2)^{3/2}}$
磁矩形式：令 $\vec{P}_m = I\pi R^2\hat{i}$，则 $\vec{B} = \dfrac{\mu_0\vec{P}_m}{2\pi(x^2+R^2)^{3/2}}$
结果：$B = \dfrac{\mu_0 I R^2}{2(x^2+R^2)^{3/2}}$，方向沿轴线（右手螺旋定则）。特例：圆心处 $(x=0)$：$B = \dfrac{\mu_0 I}{2R}$；远场 $(x\gg R)$：$B \approx \dfrac{\mu_0 I R^2}{2x^3} = \dfrac{\mu_0 P_m}{2\pi x^3}$（磁偶极子场）

轴线磁场的特征
  • ①
    圆心处 $B_0 = \dfrac{\mu_0 I}{2R}$ 最大；沿轴方向单调递减
  • ②
    $x = \pm R/\sqrt{2}$ 处为 拐点（$|dB/dx|$ 最大）
  • ③
    远场 $x\gg R$：$B \propto x^{-3}$，与 磁偶极子 行为一致
  • ④
    亥姆霍兹线圈：两个相距 $R$ 的同轴圆线圈，中心区域磁场高度均匀
例2：均匀带电旋转圆盘轴线上的磁场（$x \gg R$）

圆盘半径 $R$，面电荷密度 $\sigma$，以角速度 $\omega$ 绕轴旋转，求轴线上距圆心 $O$ 为 $x$（$x\gg R$）处 P 的磁感应强度。

思路：圆盘可看成无数同心圆电流环，每个环半径 $r$、宽 $dr$，利用例1中 $x\gg R$ 的结论
电流元：面积元 $dA = 2\pi r\,dr$，电荷 $dq = \sigma\cdot 2\pi r\,dr$，周期 $T = 2\pi/\omega$，电流 $dI = dq/T = \sigma\omega r\,dr$
磁矩元：$d\vec{P}_m = \pi r^2 dI\,\hat{n} = \pi\sigma\omega r^3 dr\,\hat{n}$
远场公式：$d\vec{B} = \dfrac{\mu_0}{2\pi}\dfrac{d\vec{P}_m}{x^3}$（$x\gg r$）
积分：$B = \displaystyle\int_0^R \frac{\mu_0}{2\pi}\frac{\pi\sigma\omega r^3}{x^3}dr = \frac{\mu_0\sigma\omega R^4}{8x^3}$
结果：$B = \dfrac{\mu_0\sigma\omega R^4}{8x^3}$（$x\gg R$），与点磁偶极子 $B\propto x^{-3}$ 一致，总磁矩 $P_m = \dfrac{1}{4}\sigma\omega\pi R^4$
交互仿真：旋转带电圆盘

拖动滑块改变旋转角速度 $\omega$，观察环形电流及轴线磁场强度变化。

ω = 5 rad/s
旋转带电圆盘 → 等效环形电流 → 轴线磁场
例3：螺线管端口处的磁场

螺线管半径 $R$，长 $L$，单位长度 $n$ 匝，通电流 $I$，求端口圆心 O 处的磁感应强度。

分割：取长度 $dx$ 的螺线管段，等效为单匝线圈，$dI = nI\,dx$
微元磁场：由圆线圈公式，$dB = \dfrac{\mu_0 nI R^2 dx}{2(x^2+R^2)^{3/2}}$
换元：令 $x = R\tan\theta$，$dx = R\sec^2\theta\,d\theta$，$(x^2+R^2)^{3/2} = R^3\sec^3\theta$
积分：$B = \displaystyle\int_0^L \frac{\mu_0 nIR^2}{2(x^2+R^2)^{3/2}}dx = \frac{\mu_0 nI}{2}\cdot\frac{L}{\sqrt{L^2+R^2}}$
结果：$B = \dfrac{\mu_0 nI}{2}\cdot\dfrac{L}{\sqrt{L^2+R^2}}$。特例：$L\gg R$（半无限长），$B = \dfrac{\mu_0 nI}{2}$；无限长螺线管内部（非端口）：$B = \mu_0 nI$
例4：有限长直载流导线的磁场

导线长 $L$，电流 $I$，P 点与导线垂直距离为 $a$，导线两端连线与 P 点到垂足的连线夹角分别为 $\theta_1$（下端）、$\theta_2$（上端）。

建系：P 作垂线交导线于 O，以 O 为原点，$y$ 轴沿导线，导线上任意点距 O 为 $y$
叉积：$d\vec{B} = \dfrac{\mu_0}{4\pi}\dfrac{-aI\,dy}{(y^2+a^2)^{3/2}}\hat{k}$
换元：$y = a\tan\theta$，积分变为 $\displaystyle\int \cos\theta\,d\theta$
结果：$B = \dfrac{\mu_0 I}{4\pi a}(\cos\theta_1 - \cos\theta_2)$
讨论： 无限长（$\theta_1=0,\theta_2=\pi$）：$B = \dfrac{\mu_0 I}{2\pi a}$； 半无限长（$\theta_1=\pi/2,\theta_2=\pi$）：$B = \dfrac{\mu_0 I}{4\pi a}$； P 在延长线上：$B = 0$
