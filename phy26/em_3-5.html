洛伦兹力公式

运动电荷 $q$ 以速度 $\vec{v}$ 在磁场 $\vec{B}$ 中运动时，受到洛伦兹力：

$$\vec{F} = q\vec{v}\times\vec{B}$$

大小：$F = qvB\sin\theta$，$\theta = \angle(\vec{v},\vec{B})$

关键性质 洛伦兹力始终垂直于速度，因此不做功（$W = \vec{F}\cdot d\vec{s} = 0$）。它只改变速度方向，不改变速率和动能。
洛伦兹力的方向：右手定则
  • ①
    四指从 $\vec{v}$ 弯向 $\vec{B}$，大拇指方向即为 $\vec{v}\times\vec{B}$ 方向
  • ②
    正电荷受力方向 = $\vec{v}\times\vec{B}$；负电荷受力方向相反
  • ③
    $\vec{v}\parallel\vec{B}$（$\theta=0°$ 或 $180°$）时：$F=0$，粒子匀速直线运动
  • ④
    $\vec{v}\perp\vec{B}$（$\theta=90°$）时：$F=qvB$，最大，粒子做匀速圆周运动
交互仿真：洛伦兹力方向与大小

拖动滑块改变速度方向角 $\theta$（$\vec{v}$ 与 $\vec{B}$ 的夹角），观察洛伦兹力大小和方向变化。

θ = 90° F/F_max = 1.000
电荷符号：
红：速度 v；蓝：磁场 B（向纸面内）；绿：洛伦兹力 F
均匀磁场中带电粒子的运动

情形一：$\vec{v}\perp\vec{B}$ — 洛伦兹力提供向心力，粒子做匀速圆周运动：

$$qvB = \frac{mv^2}{R} \implies R = \frac{mv}{qB}$$

回旋周期（与速度无关！）：$T = \dfrac{2\pi m}{qB}$；回旋频率：$f = \dfrac{qB}{2\pi m}$

情形二：$\vec{v}$ 与 $\vec{B}$ 成任意角 $\alpha$ — 将 $\vec{v}$ 分解：

  • ⊥
    $v_\perp = v\sin\alpha$：在垂直于 $\vec{B}$ 的平面内做圆周运动，半径 $R = mv_\perp/qB$
  • ∥
    $v_\parallel = v\cos\alpha$：沿 $\vec{B}$ 方向匀速直线运动（不受力）
  • 合
    合运动为螺旋线，螺距 $h = v_\parallel T = \dfrac{2\pi mv\cos\alpha}{qB}$
交互仿真：带电粒子在匀强磁场中的运动

选择初速方向（与 $\vec{B}$ 夹角 $\alpha$），观察粒子轨迹：圆周、螺旋线或直线。

α（v 与 B 夹角）= 90°
蓝色背景箭头：B 场方向（向右）；粒子轨迹实时绘制
磁聚焦原理

不同 $\alpha$ 但相同 $v_\parallel$ 的粒子，回旋周期 $T = 2\pi m/qB$ 相同，经过一个周期后，所有粒子重新聚焦到同一点 — 即磁聚焦。应用于粒子束整形、磁镜约束等。

应用：回旋加速器 粒子在两个 D 形电极间被电场加速，在磁场中做半圆轨迹，半径随速度增大而增大，最终从外缘引出。由于 $T$ 与 $v$ 无关，交变电场频率可固定（非相对论情形下）。
速度选择器

令电场力与磁场力相互平衡：$qE = qvB$，则选出速度 $v = E/B$，与电荷量无关。

原理：电场 $\vec{E}$（向上）与磁场 $\vec{B}$（垂直纸面）同时作用
平衡条件：$qE = qvB \Rightarrow v = E/B$
选速效果：仅满足该速度的粒子直线穿过，其余偏转被挡板拦截
Thomson 实验（比荷测量）

先用速度选择器得到 $v = E/B$，再令粒子进入电场加速并偏转：$\dfrac{1}{2}mv^2 = eV$，结合偏转量求出 $e/m$。

步骤1（速度选择）：$v = E_1/B_1$
步骤2（偏转）：粒子在纯电场 $E_2$ 中偏转，$y = \dfrac{eE_2 L^2}{2mv^2}$
求比荷：$\dfrac{e}{m} = \dfrac{2yv^2}{E_2 L^2} = \dfrac{2y E_1^2}{E_2 L^2 B_1^2}$
历史意义：1897年汤姆逊用此方法发现电子，测得 $e/m \approx 1.76\times10^{11}\ \text{C/kg}$

霍尔效应原理

通有电流 $I$ 的导体置于垂直磁场 $\vec{B}$ 中，在 $\vec{I}\times\vec{B}$ 方向出现横向电压 $V_H$（霍尔电压）。

导体截面：宽 $b$（电流方向），高 $h$（磁场方向垂直的横向尺寸）。

  • 1
    稳态条件：电场力 $eE_H$ = 磁场力 $evB$，即 $E_H = vB$
  • 2
    电流关系：$I = nqv\cdot(hb)$（$n$：载流子数密度），解得 $v = I/(nqhb)$
  • 3
    霍尔电压：$V_H = E_H\cdot b = vBb = \dfrac{IB}{nqh} = R_H\dfrac{IB}{h}$
霍尔系数 $R_H = \dfrac{1}{nq}$（$q$ 含符号）。$R_H > 0$：空穴导电（P 型半导体）；$R_H < 0$：电子导电（N 型）。可用于判断载流子类型和浓度。
交互仿真：霍尔效应

调节磁场强度和电流，观察载流子偏转过程和霍尔电压的建立。可切换载流子类型。

B / B₀ = 1.00 V_H / V₀ = 1.00
橙色：正载流子；蓝色：负载流子；黄色虚线：霍尔电场
霍尔效应的应用
  • ①
    磁场测量：霍尔传感器测量 $V_H$ 即可推算 $B$
  • ②
    半导体参数：确定载流子类型、浓度和迁移率
  • ③
    量子霍尔效应：二维电子气在强磁场低温下，$R_H$ 出现精确量子化台阶 $h/ne^2$，用于精密计量
  • ④
    无触点开关：霍尔开关广泛用于无刷电机换向、位置传感
