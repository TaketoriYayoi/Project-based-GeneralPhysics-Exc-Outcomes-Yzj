自感现象与自感系数

当线圈中电流变化时，其自身产生的磁通量也随之变化，从而在线圈中激发自感电动势，这一现象称为自感。

定义自感系数（自感量）$L$：

$$L = \frac{\Psi}{I}$$

其中 $\Psi = N\Phi_m$ 为全磁通（磁通量链接）。自感系数 $L$ 的单位为亨利（H），$1\,\text{H} = 1\,\text{Wb}\cdot\text{A}^{-1}$。

物理意义 $L$ 描述线圈的电磁惯性大小——$L$ 越大，电流变化时产生的自感电动势越大，电流越难以改变。$L$ 只与线圈的几何形状、匝数和磁介质有关，与电流大小无关（线性介质中）。
自感电动势

由法拉第定律，自感电动势为：

$$\varepsilon_L = -\frac{d\Psi}{dt} = -L\frac{dI}{dt}$$

负号体现楞次定律：自感电动势总是阻碍电流的变化。

  • ↑
    电流增大：$dI/dt > 0$，$\varepsilon_L < 0$，自感电动势阻碍电流增大（方向与电流反向）
  • ↓
    电流减小：$dI/dt < 0$，$\varepsilon_L > 0$，自感电动势维持电流（方向与电流同向）
  • X
    感抗：在交流电路中，$X_L = \omega L$，频率越高感抗越大，自感线圈对高频电流有阻碍作用
典型线圈的自感系数
线圈类型自感系数 $L$说明
长直螺线管$L = \mu_0 n^2 V = \mu_0 \dfrac{N^2}{l} S$$n$ 为匝密度，$V=Sl$ 为体积
同轴电缆（单位长度）$L = \dfrac{\mu}{2\pi}\ln\dfrac{R_2}{R_1}$$R_1$、$R_2$ 为内外半径，$\mu$ 为介质磁导率
环形螺线管$L = \dfrac{\mu N^2 S}{2\pi R}$$R$ 为环半径，$S$ 为截面积
互感现象与互感系数

当线圈 1 中电流 $I_1$ 变化时，其磁场穿过线圈 2，在线圈 2 中激发互感电动势，反之亦然。

定义互感系数 $M$：

$$M = \frac{\Psi_{21}}{I_1} = \frac{\Psi_{12}}{I_2}$$

其中 $\Psi_{21}$ 是线圈 1 的电流 $I_1$ 在线圈 2 中产生的全磁通，$\Psi_{12}$ 反之。互感系数 $M$ 的单位也是亨利（H），且 $M_{12} = M_{21} = M$（互易定理）。

互感电动势 $$\varepsilon_{21} = -M\frac{dI_1}{dt}, \qquad \varepsilon_{12} = -M\frac{dI_2}{dt}$$ 互感电动势的方向同样由楞次定律确定。
耦合系数与互感的上限

对于两个线圈，互感系数满足：

$$M \leq \sqrt{L_1 L_2}$$

定义耦合系数 $k = \dfrac{M}{\sqrt{L_1 L_2}}$，$0 \leq k \leq 1$。

  • 1
    全耦合（$k=1$）：两线圈完全套合，磁通量全部互链，$M = \sqrt{L_1 L_2}$。例如两同轴螺线管体积相同时。
  • 0
    零耦合（$k=0$）：两线圈磁场完全正交，互不影响，$M = 0$。
两同轴螺线管（$V_1 = V_2$） $M = \mu_0 n_1 n_2 V_2$，当 $V_1 = V_2$ 时 $M = \sqrt{L_1 L_2}$，达到全耦合。
自感的串联等效

两线圈串联时，需考虑互感的影响（顺接或反接）：

连接方式等效自感说明
顺接（磁场同向）$L = L_1 + L_2 + 2M$互感增强总自感
反接（磁场反向）$L = L_1 + L_2 - 2M$互感削弱总自感
无互感串联$L = L_1 + L_2$$M=0$ 时退化为普通串联
测量互感的方法 顺接测得 $L_+$，反接测得 $L_-$，则 $M = \dfrac{L_+ - L_-}{4}$。
RL 电路充放电仿真

电源 $\varepsilon$、电阻 $R$、电感 $L$ 串联。接通电源后，电流按指数规律上升：

$$I(t) = \frac{\varepsilon}{R}\!\left(1 - e^{-t/\tau}\right), \quad \tau = \frac{L}{R}$$

$\tau$ 称为时间常数，$\tau$ 越大，电流上升越慢（电感的"惯性"越强）。

例题 1：同轴电缆的自感（单位长度）

一电缆由两个无限长同轴圆筒状导体组成，内筒半径 $R_1$，外筒半径 $R_2$，其间充满磁导率为 $\mu$ 的磁介质。电流 $I$ 在内筒流去，在外筒流回，求单位长度的自感 $L$。

求磁场分布：由安培环路定理，两柱面之间（$R_1 < r < R_2$）：$B = \dfrac{\mu I}{2\pi r}$；$r < R_1$ 或 $r > R_2$ 处 $B = 0$。
求磁通量：取单位长度 $l=1$，面元 $dS = l\,dr = dr$：$$\Phi_m = \int_{R_1}^{R_2} B\,dr = \int_{R_1}^{R_2}\frac{\mu I}{2\pi r}\,dr = \frac{\mu I}{2\pi}\ln\frac{R_2}{R_1}$$
求自感：$L = \dfrac{\Phi_m}{I} = \dfrac{\mu}{2\pi}\ln\dfrac{R_2}{R_1}$（单位长度）
结果：$L = \dfrac{\mu}{2\pi}\ln\dfrac{R_2}{R_1}$（H/m）。磁导率 $\mu$ 越大、半径比 $R_2/R_1$ 越大，自感越大。
例题 2：两同轴螺线管的互感与自感

两同轴长直细螺线管，参数分别为：半径 $R_1 < R_2$，匝数 $N_1, N_2$，长度均为 $l$，求自感 $L_1, L_2$ 和互感 $M$。

自感 $L_1$：内管磁场 $B_1 = \mu_0 n_1 I_1$（$n_1 = N_1/l$），全磁通 $\Psi_1 = N_1 B_1 S_1 = \mu_0 n_1^2 \pi R_1^2 l \cdot I_1$，故 $L_1 = \mu_0 n_1^2 V_1$（$V_1 = \pi R_1^2 l$）
自感 $L_2$：同理，$L_2 = \mu_0 n_2^2 V_2$（$V_2 = \pi R_2^2 l$）
互感 $M$：内管电流 $I_1$ 产生的磁场 $B_1 = \mu_0 n_1 I_1$ 穿过外管截面 $S_1$（注意只穿过内管面积），$\Psi_{21} = N_2 B_1 S_1 = \mu_0 n_1 n_2 \pi R_1^2 l \cdot I_1$，故 $M = \mu_0 n_1 n_2 V_1$
特殊情形：若 $R_1 = R_2$（$V_1 = V_2 = V$），则 $M = \mu_0 n_1 n_2 V = \sqrt{L_1 L_2}$，达到全耦合 $k=1$。
结果：$L_1 = \mu_0 n_1^2 V_1$，$L_2 = \mu_0 n_2^2 V_2$，$M = \mu_0 n_1 n_2 V_1$（内管体积决定互感）。
