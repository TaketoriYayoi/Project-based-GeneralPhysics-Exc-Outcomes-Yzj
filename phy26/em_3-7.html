磁介质的分类

当磁场中存在介质时，介质被磁化，产生附加磁场，影响原磁场：$\vec{B} = \vec{B}_0 + \vec{B}'$

相对磁导率 $\mu_r = B/B_0$，磁导率 $\mu = \mu_0\mu_r$

类型μᵣ 范围特征典型材料
抗磁质$\mu_r < 1$附加磁场与外场反向，微弱排斥Cu、Bi、水、超导体（μᵣ=0）
顺磁质$\mu_r > 1$（略大于1）附加磁场与外场同向，微弱吸引Al、O₂、铂
铁磁质$\mu_r \gg 1$（可达$10^3\sim10^5$）强烈磁化，磁滞，有居里点Fe、Ni、Co、铁氧体
完全抗磁体$\mu_r = 0$完全排斥磁场（Meissner效应）超导材料
交互仿真：三类磁介质在外场中的响应

选择介质类型，观察介质内部磁场与外场的对比，以及磁力线的变化。

抗磁质：介质内磁场略小于外场，磁力线稀疏
分子磁矩与磁化强度

分子磁矩：分子/原子中电子轨道运动与自旋等效为分子电流，其磁矩称为分子磁矩 $\vec{P}_m$。

磁化强度（单位 A/m）：单位体积内分子磁矩的矢量和：

$$\vec{M} = \frac{\sum_i \vec{m}_i}{\Delta V} = \frac{\sum\vec{P}_m + \sum\Delta\vec{P}_m}{\Delta V}$$
  • 顺
    顺磁质：$\sum\vec{P}_m \gg \sum\Delta\vec{P}_m$，原有磁矩转向外场，$\vec{M}$ 与 $\vec{B}_0$ 同向
  • 抗
    抗磁质：$\sum\vec{P}_m = 0$，附加磁矩 $\Delta\vec{P}_m$ 反向于外场，$\vec{M}$ 与 $\vec{B}_0$ 反向
磁化电流

介质磁化的宏观效果：在介质表面（边缘）出现环形磁化电流 $I_s$，其面密度 $j_s = I_s/L$（A/m）。

可以证明：$|\vec{M}| = j_s$，即磁化强度等于磁化面电流密度。进一步有：

$$\oint_L \vec{M}\cdot d\vec{l} = \sum I_s$$
顺磁 vs 抗磁 顺磁质：磁化电流与螺线管传导电流同向 → 增强磁场； 抗磁质：磁化电流反向 → 削弱磁场。

有介质时的安培环路定理

有磁介质时，安培环路定理需同时考虑传导电流 $I_0$ 和磁化电流 $I_s$：

$$\oint_L \vec{B}\cdot d\vec{l} = \mu_0\!\left(\sum I_0 + \oint_L \vec{M}\cdot d\vec{l}\right)$$

整理得：$\displaystyle\oint_L\!\left(\frac{\vec{B}}{\mu_0} - \vec{M}\right)\cdot d\vec{l} = \sum I_0$

定义磁场强度 $\vec{H} = \dfrac{\vec{B}}{\mu_0} - \vec{M}$（单位 A/m），则：

$$\oint_L \vec{H}\cdot d\vec{l} = \sum I_0$$
重要性质 该式对真空和介质均成立，且只涉及传导电流，不涉及磁化电流，使计算大为简化。对各向同性均匀磁介质：$\vec{M} = \chi_m \vec{H}$，$\mu_r = 1+\chi_m$，$\vec{B} = \mu\vec{H} = \mu_0\mu_r\vec{H}$
交互仿真：有介质螺线管中 H 与 B 的分布

调节相对磁导率 $\mu_r$，对比螺线管内外的 $\vec{H}$（仅由传导电流决定，不随介质变化）和 $\vec{B} = \mu_r\mu_0\vec{H}$（随介质变化）。

μᵣ = 1.0 B/B₀ = 1.0
蓝：H（不变）；橙：B（= μᵣ × B₀，随介质变化）
例题：有介质螺绕环的磁参数

螺绕环：200匝，周长 0.1 m，截面积 $S = 5\times10^{-5}\ \text{m}^2$，通电 0.1 A，测得磁通量 $\Phi = 6\times10^{-5}\ \text{Wb}$，求相对磁导率。

磁场强度：$HL = NI \Rightarrow H = NI/L = 200\times0.1/0.1 = 200\ \text{A/m}$
磁感应强度：$B = \Phi/S = 6\times10^{-5}/(5\times10^{-5}) = 1.2\ \text{T}$
相对磁导率：$\mu_r = B/(\mu_0 H) = 1.2/(4\pi\times10^{-7}\times200) \approx 4775$
结果：$\mu_r \approx 4775$（典型铁磁质数值）
铁磁质的特点
  • ①
    高导磁性：$\mu_r \gg 1$（$10^2\sim10^5$），集中磁通量，常用于变压器、电机铁芯
  • ②
    非线性：$B$-$H$ 关系非线性，$\mu_r$ 随 $H$ 变化，不是常数
  • ③
    磁滞：$B$ 的变化落后于 $H$ 的变化，形成磁滞回线；剩余磁化强度（剩磁）和矫顽力
  • ④
    居里点：超过临界温度（居里温度）后铁磁性消失，变为顺磁质（Fe: 770°C，Ni: 358°C）

磁畴理论

铁磁质内部自发形成磁畴（spontaneous magnetization domain）：每个磁畴内部磁矩整齐排列，但不同磁畴方向随机，宏观上合磁矩为零（未磁化状态）。

施加外场后，有利方向的磁畴扩大，不利方向的磁畴缩小或翻转，宏观磁化出现。

位移电流的概念

通过某截面的位移电流等于该截面上电位移通量的时间变化率：

$$I_d = \frac{d\Phi_D}{dt} = \iint_S \frac{\partial\vec{D}}{\partial t}\cdot d\vec{S}$$

位移电流密度：$\vec{j}_d = \dfrac{\partial\vec{D}}{\partial t}$

理解方式 以平行板电容器为例：导线中流入传导电流 $I_0$，极板间虽无自由电荷流动，但 $\vec{D}$ 随时间变化，等效为"电流"连续通过。全电流（传导 + 位移）始终连续，满足电流连续性方程。
全电流定律与推广的安培环路定理

全电流 = 传导电流 $I_0$ + 位移电流 $I_d$，全电流连续：

$$\oiint_S\!\left(\vec{j} + \frac{\partial\vec{D}}{\partial t}\right)\cdot d\vec{S} = 0$$

推广的安培环路定理（麦克斯韦修正）：

$$\oint_L \vec{H}\cdot d\vec{l} = I_0 + I_d = \iint_S\vec{j}\cdot d\vec{S} + \iint_S\frac{\partial\vec{D}}{\partial t}\cdot d\vec{S}$$
深刻意义 麦克斯韦引入位移电流后，使安培定律适用于非稳恒情况。变化电场激发磁场，变化磁场激发电场，导致电磁波的存在！传播速度 $c = 1/\sqrt{\varepsilon_0\mu_0} \approx 3\times10^8\ \text{m/s}$。

例题：给电容器充电，求位移电流和极板间磁场

平行板电容器，电容 $C$，充电电流 $i = 0.2e^{-t}$（SI），$q(t=0)=0$。求（1）极板电压 $U(t)$；（2）位移电流 $I_d$。

（1）电压：$q = \displaystyle\int_0^t i\,dt = 0.2(1-e^{-t})$；$U = q/C = \dfrac{0.2}{C}(1-e^{-t})$
（2）位移电流：$I_d = \dfrac{d\Phi_D}{dt} = \dfrac{d(DS)}{dt} = S\varepsilon_0\dfrac{dE}{dt} = \dfrac{dq}{dt} = i = 0.2e^{-t}$
物理意义：$I_d = I_0 = i$（传导电流在极板处等于位移电流，全电流连续）
结果：$U(t) = \dfrac{0.2}{C}(1-e^{-t})$；$I_d = 0.2e^{-t}$（与传导电流相等！）
